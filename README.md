# Nave-S.github.io

App-Seiten für Privacy, Terms & Support. Läuft auf GitHub Pages.

## Struktur

```
/                     → Übersicht aller Apps
/style.css            → Gemeinsames Design (einmal ändern, überall wirkt)
/sqnz/privacy/        → SQNZ Datenschutz
/sqnz/terms/          → SQNZ Nutzungsbedingungen
/sqnz/support/        → SQNZ Support + FAQ
```

## Neue App hinzufügen

1. Ordner erstellen: `/neuename/privacy/`, `/neuename/terms/`, `/neuename/support/`
2. `index.html` von einer bestehenden App kopieren und anpassen
3. In `/index.html` eine neue App-Card hinzufügen (Vorlage ist im HTML-Kommentar)
4. Pushen — sofort live

## Design ändern

Nur `/style.css` bearbeiten — wirkt auf alle Seiten.
